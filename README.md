一、 功能模块：权限管理模块
功能需求
1、用户基础信息管理
2、角色基础信息管理
3、菜单基础信息管理
4、用户角色信息维护(维护用户和角色之间的关系)
5、角色菜单信息维护(维护角色和用户之间的关系)
